# React API Fetch App

A simple React application built using Vite that fetches data from a public API and displays it in a clean UI.

## Features
- Fetches data from a public API
- Displays the data in a simple user interface
- Shows loading state while fetching data
- Handles error state if API fails
- Responsive layout

## Tech Stack
- React
- Vite
- JavaScript
- CSS

## API Used
https://jsonplaceholder.typicode.com/users

## How to Run the Project

1. Clone the repository

git clone https://github.com/bhaskar-7/react-api-app.git

2. Go to the project folder

cd react-api-app

3. Install dependencies

npm install

4. Run the development server

npm run dev

5. Open the app in browser

http://localhost:5173

## Project Functionality

This application fetches user data from a public API and displays it in a card layout.  
It includes loading state while data is being fetched and error handling if the API request fails.
